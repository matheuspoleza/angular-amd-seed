## beginner-amd

Beginner AMD is a sample angularAMD app. Its provide all structure necessary, include:
- [x] [Unit Testing](#unit)
- [x] [Build System](#build)
- [x] [Editor Config](#editor)
- [ ] [Interface Testing - dalekjs](#interface)

# Install

```sh
$ bower install
```

```sh
$ npm install
```

# Getting started

```sh
$ grunt build
```

```sh
$ grunt start
```
