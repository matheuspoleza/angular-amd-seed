## beginner-amd

Beginner AMD is a sample angularAmd app. Its provide for you all structure, include:
- unit testing
- build system
- editor config
- interface testing with dalekjs
- and more ... 

# Install

```sh
$ bower install
```

```sh
$ npm install
```

# Getting started

```sh
$ grunt build
```

```sh
$ grunt start
```
